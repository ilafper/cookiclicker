# CookiClicker

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.0.6.


## Dependencia

### WAIT ON
```bash
npm install wait-on --save-dev
```

### Concurrently 

```bash
npm install concurrently --save-dev
```

## Development server

To start a local development server, run:

```bash
npm run iniciarApp
```
